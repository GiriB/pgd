News

  • July 12, 2010 The model was database updated to Pisces as of June 28, 2010 and Version 0.9.3 has been released:
    • Sidechain bond angles & lengths can now be searched, plotted, dumped, and included in statistics
    • Histograms are now displayed for selected search bins whenever plotting an attribute other than observations
    • Angles input fields now have a graphical representation of the query parameter
    • Plot is now inclusive for both start and end ranges
    • Plot bins will no longer overrun borders for some plots
    • Plot axis line up properly for all plots
    • Statistics now works for large result sets
    • Added new chis to import - His:Chi2, Pro:Chi2, Pro:Chi3
  • April 20, 2010 The model was database updated to Pisces as of Feb 13, 2010 (The latest available). The static web pages were overhauled - adding more options and information. A bug was fixed which caused incorrect values for bond lengths and angles to be calculated when a set of alturnative conformations spanned two residues.
  • January 23, 2010 The geometry library was updated to match the current Pisces library. In addition, the ability to filter and search for side chain χ angles was implemented.
